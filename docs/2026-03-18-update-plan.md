**파일: `app/cathedral/page.tsx`**

`REFINE_COMPONENTS` 배열 (라인 19~24)의 `amount` 값 수정:
```ts
const REFINE_COMPONENTS = [
  { itemId: '66102007', name: '운명의 파괴석 결정', icon: '...', amount: {실제값} }, // 현재 1000
  { itemId: '66102107', name: '운명의 수호석 결정', icon: '...', amount: {실제값} }, // 현재 1000
  { itemId: '66110226', name: '위대한 운명의 돌파석', icon: '...', amount: {실제값} }, // 현재 1000
  { itemId: '66130143', name: '운명의 파편 주머니(대)', icon: '...', amount: {실제값} }, // 현재 1000
```

> 이 값만 바꾸면 자동으로:
> - 상자 총 가치 = Σ(수량 × 거래소 단가) (체크박스 포함/제외)
> - 은총 1개 가치 = 상자 총 가치 ÷ 60
> - UI에 실시간 표시
